РАСШИРЕННОЕ РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ ПРОГРАММЫ PlainInstead
Данное руководство описывает работы приложения более подробно и предназначено для опытного пользователя. Программа является оболочкой оригинального интерпретатора TADS, разработанного Майклом Робертсом. 

ПАПКИ И ФАЙЛЫ ПРОГРАММЫ
* games - содержит библиотеку игр. Вы можете самостоятельно добавить туда скомпилированную игру формата gam и дополнить описание в файле info.ini. Смотрите примеры описаний других игр.
* rtads_au - консольная среда разработки игр на языке RTADS. Подробнее смотрите readme.txt в данной папке
* saves - каталог сохранений для игр. Игры сохраняются в бинарном формате, который не подходит к другим версиям игр или даже к другой версии той же самой игры.
* temp - папка для хранения временных файлов интерпретатора и отчетов о прохождении. Сохранение лога прохождения начинается по команде ОТЧЕТ, остановка записи по команде ОСТОТЧЕТ, файл результата c именем log.txt.
* settings.ini - файл с настройками, можно редактировать вручную или создавать несколько вариантов настроек. Область [MainFrame] отвечает за положение и размеры окна, менять не нужно. [main] хранит основные настройки.

КОММЕНТАРИИ К НАСТРОЙКАМ
Файл->Сбросить все настройки: удаляет файл settings.ini и при повторном включении создаёт заново
Просмотр->Настройка
Озвучивать новый текст через программу экранного доступа - если кого-то не устраивает автоматическое озвучивание, то можно снять данную галочку и самостоятельно передвигаться курсором к полю вывода.
Перемещать фокус на поле текстового вывода после ввода команды - позволяет облегчить процесс навигации без автоматического звучания. Если снять предыдущую галочку и установить эту, то после ввода сразу попадаем в окно вывода и можем двигаться курсором.

ДОПОЛНИТЕЛЬНЫЕ ВОЗМОЖНОСТИ
* Если начать набирать текст в окне вывода, то курсор автоматически перескочит в поле ввода.
* Можно запустить оболочку сразу с игрой через параметр командной строки, например PlainInstead games\zz.gam
* Для того, чтобы повторяющиеся команды не попали в историю, надо нажать CTRL+2, чтобы снова вернуть запись истории CTRL+1
